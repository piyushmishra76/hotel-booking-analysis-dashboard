# hotel-booking-analysis-dashboard
This project explores hotel booking data using Microsoft Excel to analyze customer behavior, pricing trends, and cancellation patterns.
---

## 📊 Project Overview

The objective of this project is to extract meaningful insights from hotel booking data and present them through an interactive dashboard.
This project focuses on analyzing hotel booking data to understand customer behavior, pricing patterns, and cancellation trends. The goal is to extract meaningful insights and build an interactive dashboard using Microsoft Excel.


---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

---

## 📌 Key Insights

### 🔹 Booking Trends
- Booking demand varies across months indicating seasonal patterns.

### 🔹 Cancellation Behavior
- Higher lead time leads to higher cancellation probability.
- Deposit type significantly affects cancellation rates.

### 🔹 Customer Behavior
- Repeat guests are more reliable (lower cancellation rate).
- Repeat guests tend to have lower ADR due to loyalty benefits.

### 🔹 Pricing Insights (ADR)
- Most bookings fall within mid-range pricing.
- Distribution is right-skewed with a few high-value bookings.

### 🔹 Room Allocation
- Upgrades often come at lower ADR.
- Downgrades may still have higher ADR, indicating pricing is fixed at booking.

### 🔹 Customer Composition
- Majority bookings are adults-only.
- Family bookings are fewer.
